CountdownPlugin

The plugin shows the remaining days until a configured target date.

REST API

Get target date and target day description.

GET {{ORIGIN}}/rest/api/v1/display/uid/<PLUGIN-UID>/countdown
  • PLUGIN-UID: The plugin unique id.

Set target day and target day desription.

POST {{ORIGIN}}/rest/api/v1/display/uid/<PLUGIN-UID>/countdown?day=<DAY>&month=<MONTH>&year=<YEAR>&plural=<PLURAL>&singular=<SINGULAR>
  • PLUGIN-UID: The plugin unique id.
  • DAY: The target date day.
  • MONTH: The target date month.
  • YEAR: The target date year (YYYY format, e.g. 2020)
  • PLURAL: The unit in plural form, e.g. DAYS.
  • SINGULAR: The unit in singular form, e.g. DAY.
