OpenWeatherPlugin

The plugin shows weather information provided by https://openweathermap.org/.

REST API

Get api key and city ID.

GET {{ORIGIN}}/rest/api/v1/display/uid/<PLUGIN-UID>/weather
  • PLUGIN-UID: The plugin unique id.

Set OpenWeather API key and city ID.

POST {{ORIGIN}}/rest/api/v1/display/uid/<PLUGIN-UID>/weather?apiKey=<API-KEY>&cityId=<CITY-ID>
  • PLUGIN-UID: The plugin unique id.
  • API-KEY: The OpenWeather api key.
  • CITY-ID: The OpenWeather city ID.
