JustTextPlugin

The plugin shows only text on the whole display.

REST API

Text

POST {{ORIGIN}}/rest/api/v1/display/uid/<PLUGIN-UID>/text?show=<TEXT>
  • PLUGIN-UID: The plugin unique id.
  • TEXT: The text to show on the display.
